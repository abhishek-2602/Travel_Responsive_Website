# Travel_Responsive_Website
  
A simple, clean, and fully responsive **Travel Website** built using **HTML, CSS, and JavaScript**.  
This project showcases a modern UI with sections like Home, Destinations, About, and Contact, along with a working mobile navigation menu.

---

##  Features

###  Fully Responsive Layout  
Works smoothly on mobile, tablet, and desktop using media queries.

###  Modern Navigation Bar  
- Sticky navbar  
- Hover effects  
- Mobile-friendly burger menu

###  Hero / Home Section  
Large background image with title, description, and CTA button.

###  Destinations Section  
A responsive grid layout showing popular destinations using Unsplash images.

###  About Section  
Clean text section describing the travel brand.

###  Contact Form  
Simple form with Name, Email, Message fields.

###  Mobile Burger Menu  
JavaScript-powered menu toggle for small screens.

---

##  Technologies Used

| Technology | Purpose |
|-----------|---------|
| **HTML5** | Structure of the website |
| **CSS3** | Styling, layout, responsiveness |
| **JavaScript (Vanilla JS)** | Burger menu toggle |
